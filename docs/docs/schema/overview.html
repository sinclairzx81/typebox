Schema

Native JSON Schema Validation and Inference

Overview

The TypeBox Value and Compile submodules are built over a more low level JSON Schema compliant validation system that supports Draft 3 to 2020-12. This validation system is fully decoupled from Type and Value submodules and is designed to be a lightweight, high performance and spec compliant alternative to Ajv.

import Schema from 'typebox/schema'

Example

The following uses the Schema module to Parse a value using JSON Schema

// ------------------------------------------------------------------
// Schema
// ------------------------------------------------------------------

const T = {
  type: 'object',
  required: ['x', 'y', 'z'],
  properties: {
    x: { type: 'number' },
    y: { type: 'number' },
    z: { type: 'number' }
  }
} as const

// ------------------------------------------------------------------
// Parse
// ------------------------------------------------------------------

const R = Schema.Parse(T, {                         // const R: {
  x: 1,                                             //   x: number,
  y: 2,                                             //   y: number,
  z: 3                                              //   z: number
})                                                  // } = ...
