# Security Policy

Details for submitting security related issues

## Supported Versions

The following versions are active and receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report suspected security vulnerabilities privately through GitHub Security Advisories:

https://github.com/sinclairzx81/typebox/security/advisories/new
